This project includes a Streamlit dashboard for interactive exploration and visualization of solar data across West African countries.

### Features

- **Country Selection**: Multi-select widget to choose countries for analysis
- **Multiple Metrics**: Compare GHI, DNI, DHI, Temperature, and Wind Speed
- **Interactive Visualizations**: Box plots, bar charts, and violin plots
- **Performance Ranking**: Automated ranking of countries by solar potential
- **Real-time Metrics**: Key performance indicators and statistics
